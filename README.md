# Car Booking Dashboard Analysis (Excel)

## 📊 Project Overview
This project focuses on analyzing taxi booking data across major cities (**Chennai, Delhi, and Mumbai**) to track revenue and service quality.

## 🧹 Data Cleaning
* **Standardized fare data** for accurate financial reporting.
* **Organized pickup locations** to compare city-level performance.
* **Aggregated driver ratings** to measure customer satisfaction.

## 📈 Analysis Performed
* **Calculated Total Revenue (₹2,250)** across different metro branches.
* **Created Pivot Tables** to compare payment methods and city performance.
* **Identified top-performing drivers** and regions using average ratings.

## 📊 Dashboard Features
* **KPI Metrics:** Tracking Total Revenue, Top City, and Highest Avg Rating.
* **Revenue by City:** Visual bar charts comparing city-wise sales.
* **Payment Distribution:** Breakdown of UPI, Cash, and Card usage.
* **Interactive Slicers:** Added **Driver Rating filters** for dynamic data viewing.

## 🔍 Key Insights
* **Mumbai** generates the highest revenue among all cities.
* **Chennai** leads in service quality with the highest average driver rating.
* **UPI** is the most preferred payment method, showing high digital adoption.

## 🛠 Tools Used
* **Microsoft Excel:** (Pivot Tables, Slicers, Charts, KPI Modeling).
* **GitHub:** Portfolio Management.
